Mastercard Cybersecurity Simulation

Insights into identifying and addressing security threats

Overview

In July 2024, I completed the Mastercard Cybersecurity Simulation on Forage. This program provided a comprehensive understanding of working on Mastercard's Security Awareness Team. I tackled real-world security challenges, identifying and mitigating potential threats to the organization.

Key Highlights

  • Identified and reported security threats, including phishing attempts
  • Analyzed business areas needing enhanced security training
  • Implemented security awareness training courses and procedures

This simulation emphasized strategic thinking and effective communication, enabling me to design and present robust security solutions.

Skills Gained

  • Security Awareness Training
  • Problem Solving
  • Strategy
  • Data Analysis
  • Communication
  • Data Presentation
